# Lifelapse-Game
A prototype of a 2D side-scrolling shooter game designed to teach financial responsibility and resource management concepts to secondary students.

This was created as part of the Ä EMMA financial literacy gamification contest in 2019 as well as for the ARI3216 study unit at the University of Malta the report for this project can be seen in "ARI3216 Report.pdf" whilst the "PitchPresentation.mp4" file contains a dubbed presentation of the created prototype.
